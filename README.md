# Japanese translation updates of BZ mods

:warning: This MOD will be incompatible with BZ mods newer than their versions as of Oct 2023.

![https://mods.factorio.com/mod/bz-ja-updates](https://img.shields.io/badge/dynamic/json.svg?label=Downloads&url=https%3A%2F%2Fmods.factorio.com%2Fapi%2Fmods%2Fbz-ja-updates&query=%24.downloads_count&colorB=%23b548aa)

## Summary

This MOD updates (corrects) some Japanese translation of BZ mods.

Though I have made some PRs for these updates, the development of BZ mods appears inactive since May 2023.  So add this MOD for interim fixes.

It also contains Japanese translation of Noble Metals (bzgold).
